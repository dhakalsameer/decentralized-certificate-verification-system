# Decentralized Certificate Verification System

A blockchain-based system for issuing and verifying academic and professional certificates. This project aims to eliminate certificate fraud and provide a secure, immutable, and easily verifiable way to manage credentials.

## Features

- **Decentralized Issuance**: Educational institutions and organizations can issue certificates directly to the blockchain.
- **Instant Verification**: Anyone can verify the authenticity of a certificate without contacting the issuing authority.
- **Tamper-Proof**: Certificates are stored using blockchain technology, ensuring they cannot be altered or forged.
- **User Privacy**: Users have control over who can view their certificates.
